## Overview

A high-performance Wordle solver that leverages NVIDIA CUDA for parallel computing to rapidly eliminate candidate words.

## Technical Details

- Implements parallel word elimination using CUDA kernels
- Optimized for GPU memory access patterns
- Significantly faster than CPU-based solvers for large word lists
