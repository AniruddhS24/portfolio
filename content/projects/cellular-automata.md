A visualization tool for generating and exploring elementary cellular automata patterns.

- Built with Java and Wolfram Mathematica
- Supports various cellular automaton rules
- Visual pattern generation and analysis
- Investigated Rule 45 for pattern periodicity during Wolfram Research internship
